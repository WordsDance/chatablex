**ChatableX（小蜜蜂智能体）** — 桌面 AI Agent。让不会写代码的人，用自然语言造出专属工具；让需要掌控感的人，看清 Agent 的每一步推理与执行。

## 为什么选择 ChatableX

**零代码造 Tool**

> 描述即创造。在对话中表达意图，即可生成并迭代专属工具——无需脚本、无需脚手架、无需维护提示词工程。

**Agentic Harness — 执行全程可见**

> 推理链、执行计划、工具调用的输入输出与环境状态，于独立面板完整呈现。可定位、可中断、可回溯；关键动作经确认后再继续（2PC）。Agent 不再是黑盒。

**录制即 Tool**

> 在界面上示范一次，Agent 将操作路径蒸馏为可复用的 Tool。以演示代替编码，以沉淀代替重复劳动。

**本地桌面工作站**

> 对话、数据与工作流运行于本地。所造工具驻留于你的环境，可持续调用，亦可交付他人安装使用。
